# my-marketplace

my 个人 Claude Code 插件市场。

存放 Claude Code plugins 的元数据配置。

## 安装 Marketplace

在 Claude Code 中运行以下命令添加此 Marketplace：

```bash
/plugin marketplace add my-marketplace https://github.com/uvidswqkscr89/my-marketplace.git
```

## 可用插件

### my-skills

包含 PRD 工作流和测试用例生成的核心技能库。

**安装:**

```bash
/plugin install my-skills@my-marketplace
```
